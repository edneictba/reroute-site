# REROUTE Portal - Componentes Visuais

Esta sprint usa componentes estaticos baseados em classes CSS reutilizaveis.

Componentes definidos em `src/portal/styles/portal.css`:

- PortalShell
- PortalHeader
- PortalSidebar
- PortalTopbar
- PortalFooter
- PageHeader
- Card
- MetricCard
- StatusBadge
- Button
- Input
- Select
- Modal
- EmptyState
- LoadingState
- ErrorState
- Breadcrumb
- UserMenu
- MobileNavigation

Quando o Portal evoluir para componentes JavaScript ou framework, estes nomes devem orientar a componentizacao.
